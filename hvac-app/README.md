# SpeedyHVAC - Field Service Management Platform

## Overview
SpeedyHVAC is a comprehensive field service management platform specifically designed for HVAC service companies. It provides a complete solution for managing technicians, service requests, customers, equipment, and invoicing.

## Key Features

### Dashboard
- Real-time overview of business metrics
- Service request status tracking
- Technician availability and workload visualization
- Revenue and performance analytics
- Interactive charts and graphs

### Work Order Management
- Create and assign service requests
- Track service request status
- Schedule appointments
- Manage priority levels
- Attach documentation and photos

### Technician Mobile Interface
- Mobile-optimized interface for field technicians
- Real-time job assignment notifications
- GPS navigation to customer locations
- Digital service forms and checklists
- Parts and inventory management
- Time tracking and job status updates

### Customer Management
- Comprehensive customer database
- Service history tracking
- Equipment installation records
- Maintenance contract management
- Customer communication tools

### Equipment Tracking
- Equipment inventory management
- Maintenance history records
- Warranty information tracking
- Part replacement scheduling
- Equipment performance monitoring

### Invoicing System
- Automated invoice generation
- Payment processing
- Financial reporting
- Tax calculation
- Integration with accounting systems

## Technical Architecture
- React frontend with TypeScript
- Responsive design for desktop and mobile
- RESTful API backend
- Real-time data synchronization
- Secure authentication system
- Cloud-based deployment

## Deployment
This application is configured for easy deployment to Vercel:

1. Connect your GitHub repository to Vercel
2. Set the root directory to `hvac-app`
3. Configure the build settings:
   - Framework: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Deploy the application

## Development
To run the application locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
